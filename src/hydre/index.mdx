# Hydre

* C'est la bannière derrière laquelle se range tous nos services.
* C'est un ensemble de normes pour communiquer.
* C'est le nom le plus marrant de la terre.
* C'est un truc conçu sur RabbitMQ.

Hydre communique avec un format JSON.
La structure de ce JSON est définie principalemenet par cette documentation (voir [Models](/hydre/models)).
La partie sur les données est propre à un service, elle ne sera donc pas documenté ici.
